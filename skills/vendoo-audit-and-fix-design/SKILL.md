# Audit & Fix Design — Vendoo

Identify and correct UX and design inconsistencies across a Figma flow.

## Instructions

1. Review screens for:
   - UX clarity
   - Visual hierarchy
   - Accessibility (contrast, tap targets)
2. Identify:
   - Inconsistent spacing
   - Misaligned elements
   - Incorrect components
3. Fix issues directly:
   - Align layouts
   - Normalize spacing
   - Improve readability
4. Ensure:
   - Mobile usability (thumb zones)
   - Clear primary actions
5. Validate flows end-to-end.

## Examples

**Input:** “Audit this listing flow”

**Output:**
- Fixed spacing inconsistencies
- Improved CTA hierarchy
- Better readability and structure

## Common edge cases

- Overcrowded screens → simplify
- Weak hierarchy → reinforce primary action
- Inconsistent patterns → standardize
